## Requirements

- Yarn
- Node `$(cat .node-version)`

## Getting started

```
$ git clone https://github.com/pine/meteorshower-app.git
$ cd meteorshower-app

$ yarn
$ yarn start     # for development
$ yarn run build # for production
```

## Test

```
$ yarn test
```
